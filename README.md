# Atom Simulator
## Needed Rescources:
* Insure that pygame and numpy is downloaded
  * Open your terminal
  * For mac OS, input python3 -m pip install pygame
  * For Windows, input python -m install pygame
  * For mac OS, input pip3 install numpy
  * For Windows, input -m pip install numpy
* If pygame and numpy is installed, then the application should be ready to be ran from either your terminal or an IDE of your choice
  * For VScode, defalt settings does not have "ExecuteInFileDir" enabled, turn it on through your settings, or the game likely wont run
## Description:
The Atom Simulator is a simulator that attempts to use a physics-based approach to simulate interactions between atoms and chemical reactions. The user will be able to input different atoms of different masses and will be able to see the simulation naturally generate interaction between the placed atoms based entirely on physics with no hard-coded chemistry rules.
